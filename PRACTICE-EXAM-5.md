## Question 61

A company runs an application on EC2 instances that processes sensitive customer data. The company needs to ensure that all data at rest is encrypted. The encryption keys must be rotated annually and the company must maintain full control over the keys.

Which solution meets these requirements?

- **A.** Use AWS KMS with AWS managed keys (AWS KMS keys).
- **B.** Use AWS KMS with customer managed keys and enable automatic annual rotation.
- **C.** Use S3 server-side encryption with S3 managed keys.
- **D.** Use a third-party encryption tool with keys stored in S3.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: B**

AWS KMS customer managed keys (CMKs) give you full control over the key lifecycle, including rotation policies. You can enable automatic key rotation, which rotates the key material annually. CMKs also provide an audit trail of key usage through CloudTrail integration.

</details>

## Question 62

A company has a legacy application that uses a local message broker. The company is migrating to AWS and needs a managed message broker service that supports industry-standard APIs and protocols, requires minimal reconfiguration of the existing application, and provides high availability.

Which AWS service meets these requirements?

- **A.** Amazon SQS
- **B.** Amazon SNS
- **C.** Amazon MQ
- **D.** Amazon Kinesis

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: C**

Amazon MQ is a managed message broker service for Apache ActiveMQ and RabbitMQ. It supports industry-standard APIs and protocols (JMS, AMQP, MQTT, OpenWire, STOMP), making it compatible with existing applications with minimal reconfiguration. It provides high availability with multi-AZ deployments.

</details>

## Question 63

A company is deploying a web application globally. Users from different geographical locations report slow response times. The application consists of dynamic content generated by EC2 instances and static assets. The company wants to improve performance without modifying the application code.

Which solution meets these requirements?

- **A.** Use Amazon CloudFront with the ALB as origin for dynamic content and S3 for static content.
- **B.** Deploy the application in each Region using Route 53 geolocation routing.
- **C.** Use AWS Global Accelerator with endpoints in each Region.
- **D.** Increase EC2 instance sizes in the primary Region.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

Amazon CloudFront can cache both dynamic and static content at edge locations worldwide. Using an ALB as the origin for dynamic content allows CloudFront to serve personalized content while reducing load on the origin servers. This requires no application changes and provides the best performance improvement.

</details>

## Question 64

A company has a batch processing workload that runs nightly. The workload processes approximately 500 GB of data and typically takes 4 hours to complete. The workload can tolerate interruptions but must complete before morning. The company wants to minimize compute costs.

Which solution meets these requirements at the LOWEST cost?

- **A.** Use EC2 On-Demand instances for the batch processing.
- **B.** Use EC2 Reserved Instances for the batch processing.
- **C.** Use EC2 Spot Instances with a diversified fleet and checkpointing.
- **D.** Use AWS Lambda to process the data in parallel.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: C**

EC2 Spot Instances can provide up to 90% cost savings compared to On-Demand pricing. Since the workload can tolerate interruptions and has a flexible completion window, Spot Instances are ideal. Using a diversified fleet across multiple instance types and Availability Zones reduces interruption probability, while checkpointing allows the job to resume if interrupted.

</details>

## Question 65

Which combination of AWS services meets these requirements?

- **A.** Use AWS CodePipeline, CodeBuild, and CodeDeploy.
- **B.** Use Jenkins on EC2, ECR, and ECS.
- **C.** Use GitHub Actions with self-hosted runners.
- **D.** Use AWS CodeStar with custom deployment scripts.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

AWS CodePipeline orchestrates the CI/CD workflow. AWS CodeBuild compiles source code, runs tests, and produces software packages. AWS CodeDeploy automates deployments to ECS. All three services are fully managed, integrate seamlessly, and require minimal operational overhead.

## Question 66

A company is designing an IoT solution that collects telemetry data from millions of devices. The data needs to be ingested in real-time, stored for long-term analysis, and made available for real-time dashboards. The solution must scale automatically as the number of devices grows.

Which combination of AWS services meets these requirements?

- **A.** Use AWS IoT Core for ingestion, Kinesis Data Streams for real-time processing, S3 for storage, and OpenSearch for dashboards.
- **B.** Use API Gateway for ingestion, Lambda for processing, and DynamoDB for storage.
- **C.** Use SQS for ingestion, EC2 for processing, and RDS for storage.
- **D.** Use MQTT broker on EC2, S3 for storage, and Athena for analysis.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

AWS IoT Core provides secure, bidirectional communication with millions of IoT devices. Kinesis Data Streams handles real-time data streaming and processing. S3 provides durable, scalable storage for long-term analysis. OpenSearch Service enables real-time dashboards and analytics. All services automatically scale with the workload.

</details>

## Question 67

A company has a regulatory requirement to retain database backups for 7 years. The backups are rarely accessed but must be available within 24 hours when needed. The company wants to minimize storage costs.

Which solution meets these requirements at the LOWEST cost?

- **A.** Store backups in S3 Standard with versioning enabled.
- **B.** Store backups in S3 Glacier Deep Archive.
- **C.** Store backups on EBS volumes with snapshots.
- **D.** Store backups in Amazon RDS automated backups.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: B**

S3 Glacier Deep Archive is the lowest-cost storage class in AWS, designed for long-term retention of data that is rarely accessed. With retrieval times of up to 24 hours and storage costs significantly lower than other options, it's ideal for compliance archives that need to be retained for years.

</details>

## Question 68

Which AWS service meets these requirements with the LEAST operational effort?

- **A.** Use Amazon Kinesis Analytics to process and analyze the streaming data.
- **B.** Use Amazon EMR with Spark Streaming.
- **C.** Use EC2 instances with custom stream processing software.
- **D.** Use AWS Glue with streaming ETL jobs.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

Amazon Kinesis Data Analytics (now part of Managed Service for Apache Flink) allows you to process and analyze streaming data using SQL or Apache Flink. It's a fully managed service that automatically scales to handle millions of events per day with minimal operational effort.

## Question 69

A company is migrating a .NET application to AWS. The application uses Windows authentication and needs to be integrated with the company's existing Active Directory. The company wants to minimize changes to the application.

Which solution meets these requirements?

- **A.** Deploy the application on EC2 instances and join them to the domain using AWS Directory Service AD Connector.
- **B.** Rewrite the application to use IAM authentication.
- **C.** Use Amazon Cognito for user authentication.
- **D.** Deploy the application on Lambda with .NET Core.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

AWS Directory Service AD Connector acts as a proxy to your existing on-premises Active Directory. EC2 instances can be domain-joined to the directory, allowing the application to use Windows authentication without modification. This provides seamless integration with existing AD infrastructure.

</details>

## Question 70

A company is building a document sharing platform. Users upload documents that need to be virus scanned before being available for download. The scanning should not block the upload process, and users should receive notification if a document is infected.

Which solution meets these requirements with the LEAST operational complexity?

- **A.** Use S3 event notifications to trigger a Lambda function that performs virus scanning using a third-party library.
- **B.** Use EC2 instances with antivirus software to poll S3 for new documents.
- **C.** Use AWS WAF to scan uploaded documents.
- **D.** Use Amazon Macie to detect malicious content.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

S3 event notifications automatically trigger Lambda functions when documents are uploaded. The Lambda function can use third-party antivirus libraries or integrate with scanning services. This serverless approach doesn't block uploads, can send notifications via SNS, and has minimal operational complexity.

</details>

## Question 71

A company has a requirement to encrypt all data at rest. The company wants to use its own hardware security modules (HSMs) to generate and store encryption keys, but wants to avoid managing the HSM infrastructure in AWS.

Which solution meets these requirements?

- **A.** Use AWS KMS with customer managed keys.
- **B.** Use AWS CloudHSM with client-side encryption.
- **C.** Use AWS KMS with external key store (XKS) linked to on-premises HSMs.
- **D.** Use S3 bucket encryption with S3 managed keys.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: C**

AWS KMS External Key Store (XKS) allows you to use your own on-premises HSMs to generate and store key material while using AWS KMS for key orchestration. This gives you control over key generation and storage without managing HSM infrastructure in AWS.

</details>

## Question 72

A company operates an ecommerce site that experiences flash sales events where traffic increases by 100x for short periods. The site uses an Auto Scaling group but instances take 5 minutes to become healthy. During flash sales, the existing instances become overloaded before new instances are ready.

Which solution should the solutions architect recommend?

- **A.** Implement scheduled scaling to launch instances before flash sales.
- **B.** Use predictive scaling based on historical data.
- **C.** Maintain a warm pool of instances ready to handle traffic spikes.
- **D.** Use larger instance types that can handle the full traffic load.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: C**

EC2 Auto Scaling warm pools allow you to maintain a pool of pre-initialized instances ready to quickly serve traffic. When a scale-out event occurs, instances from the warm pool can immediately transition to the Auto Scaling group and start serving traffic, eliminating the 5-minute startup delay.

</details>

## Question 73

A company is designing a system to store and retrieve medical images. The system needs to provide sub-second retrieval for recent images and must retain all images for at least 7 years due to regulatory requirements. The access patterns are unpredictable.

Which solution meets these requirements MOST cost-effectively?

- **A.** Use S3 Standard for all images with lifecycle policies.
- **B.** Use S3 Intelligent-Tiering with Object Lock enabled.
- **C.** Store recent images in EFS and archive to Glacier after 30 days.
- **D.** Use S3 Glacier Instant Retrieval for all images.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: B**

S3 Intelligent-Tiering automatically optimizes storage costs based on access patterns without performance impact or operational overhead. It provides sub-second access for frequently accessed objects and cost savings for rarely accessed objects. S3 Object Lock ensures regulatory compliance with the 7-year retention requirement.

</details>

## Question 74

A company needs to share files between multiple VPCs in different AWS Regions. The files must be accessible using standard file system protocols (NFS/SMB) and must maintain strong consistency across all access points.

Which solution meets these requirements?

- **A.** Use VPC peering and deploy file servers in each VPC.
- **B.** Use AWS Transit Gateway and Amazon FSx for Windows File Server.
- **C.** Use Amazon EFS with VPC peering between all VPCs.
- **D.** Use AWS DataSync to replicate files between Regions.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: C**

Amazon EFS provides a regional file system that can be accessed from multiple VPCs using VPC peering or Transit Gateway. It supports NFSv4.1 protocol, provides strong consistency, and automatically replicates data across Availability Zones. EFS is accessible from any VPC that has network connectivity to the file system.

</details>

## Question 75

A company is building a data processing pipeline that needs to transform JSON data into Parquet format, partition the data by date, and make it available for analytics queries. The volume of data is 50 GB per day and the solution must be serverless.

Which combination of AWS services meets these requirements?

- **A.** Use AWS Glue to transform data and Athena to query it.
- **B.** Use EMR Serverless to transform data and Redshift Spectrum to query it.
- **C.** Use Lambda to transform data and DynamoDB to store it.
- **D.** Use Kinesis Data Analytics to transform data and OpenSearch to query it.

<details>
<summary><strong>Reveal Answer</strong></summary>

**Correct Answer: A**

AWS Glue is a serverless data integration service that can transform JSON to Parquet format and partition data. It's fully managed and scales automatically. Amazon Athena can directly query the transformed Parquet data in S3, providing a complete serverless analytics solution.

</details>
